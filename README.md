# BackStopJS Starter Kit

[BackStopJS](https://github.com/garris/BackstopJS)のスターターキット

## Usage

ZIPファイルを[ダウンロード](https://github.com/frontainer/backstop-starter-kit/archive/master.zip)して展開

### 必要な環境を構築

```shell
npm run ready
```

### テストを開始

```
npm run test
```

### 比較元キャプチャの更新

```
npm run ref
```

### 新しいコンフィグファイルの作成

```
npm run config
```
